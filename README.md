# Fred

Fred is a Discord bot that automatically fetches and posts Epic Games free game offers.

## How to use

• Add the bot to your server using this link:
https://discord.com/oauth2/authorize?client_id=1461741762547810437

• Set up a channel for free games using the /confirmchannelcommand or manually.
• View all available commands by typing /commands.

## How to run your own instance

Create a .env file containing:

Your Discord ID

Your Discord bot token

Your Epic Games API key (RapidAPI)

Install all required Python packages from requirements.txt.

Run the bot using the .vbs file to keep it running in the background.

To shut down the bot, use the /shutdown command.
